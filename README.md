# Mohamed Karim Mahmoud - Professional Portfolio

This is a premium React portfolio website built with Vite, Tailwind CSS, and Framer Motion. It features a dark theme with blue glow accents, designed for professional data analysts and developers.

## Features
- **Modern React**: Functional components with hooks.
- **Tailwind CSS**: Custom dark theme with navy/black palette.
- **Framer Motion**: Smooth entrance and hover animations.
- **Glassmorphism**: Elegant UI components with backdrop blur.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Typing Animation**: Dynamic hero section highlighting key roles.

## Prerequisites
- **Node.js** (v18 or higher recommended)
- **npm** or **yarn**

## Installation

1. Navigate to the project directory:
   ```bash
   cd mohamed-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the App

Start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:5173`.

## Building for Production

To create a production-ready bundle:
```bash
npm run build
```
The output will be in the `dist` folder.
